## Setup

1. Add virtual environment
```sh
python -m venv .venv
```

2. Switch to virtual environment
```sh
.venv\Scripts\activate.bat
```

3. Install libraries
```sh
pip install -r requirements.txt
```

## Run

1. Switch to virtual environment
```sh
.venv\Scripts\activate.bat
```

2. Run
```sh
python main.py
```
